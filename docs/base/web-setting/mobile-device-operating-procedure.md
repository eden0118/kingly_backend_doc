# 行動裝置使用流程

此功能可協助使用者將百科網站快速建立為捷徑並新增至手機桌面。

## Android 操作說明

1. 進入百科網站後，點選右上角「⋮」功能選單。
2. 選擇「加入主畫面」。
3. 點選「建立捷徑」。
4. 編輯捷徑名稱後，點擊新增。
5. 再次點擊自動新增，即可在手機桌面看到捷徑。

![alt text](img/android-share.png)

## Iphone 操作說明

1. 進入百科網站後，點選下方分享icon。
2. 選擇「加入主畫面」。
3. 編輯捷徑名稱後，點擊新增，即可在手機桌面看到捷徑。
