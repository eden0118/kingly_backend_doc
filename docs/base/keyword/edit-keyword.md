# 編輯關鍵字名稱

可隨時編輯修改關鍵字名稱，但名稱不可重複。

:::warning
關鍵字名稱修改並不會影響關聯。 <br/>
舉例來說，ABCD 四篇文章設定了「痘痘」這個關鍵詞，把「痘痘」變更為「青春痘」，那與 ABCD 關聯依然存在，檢視文章列表或文章資訊頁會看到原先「痘痘」關鍵字全部改為「青春痘」，並不需要額外重新設定。
:::

### 操作流程

1. 點擊欲修改「關鍵字名稱」，進入編輯頁面。
   ![編輯關鍵字名稱](img/edit-keyword-01.png)

2. 編輯頁面中可執行以下設定：
    - 編輯關鍵字名稱：可隨時編輯修改關鍵字名稱，但名稱不可重複。
    - 新增多語系關鍵字名稱：可新增簡體中文與英文關鍵字名稱。
    - 查看關聯內容列表：顯示與該關鍵字關聯的內容列表，點擊內容主題可前往該文章的編輯頁面。
    - 查看關聯媒體列表：顯示與該關鍵字關聯的媒體列表，點擊圖片名稱可前往該圖片的編輯頁面。
    - 查看關聯課程列表：顯示與該關鍵字關聯的課程列表，點擊課程名稱可前往該課程的編輯頁面。
      ![編輯關鍵字名稱](img/edit-keyword-02.png)

3. 點擊右上角 `保存`，系統會判斷是否與現有關鍵字有重複。

    3.1 若編輯後的名稱有重複，會儲存失敗，回復成修改前。
    ![編輯關鍵字名稱](img/edit-keyword-error.png)

    3.2 檢測無重複，即變更成功。
    ![編輯關鍵字名稱](img/edit-keyword-success.png)

4. 保存成功後，系統將依多語系設定狀態更新顯示樣式；當關鍵字已設定多語系名稱時，對應的語系按鈕將顯示為綠色。
