# 運動課程

### 新增/編輯運動課程

說明課程資訊各欄位的限制與說明。

- 名稱：不可重複
- 課程說明
- 圖檔
- 關鍵字
- MET 代謝當量
- 綁定器材

    最多可設定三項器材。

    僅能從器材清單內選擇。

- 循環模式設定
    - 組數重複：A >> A >> A >>　 B >> B >> B >>　 C >> C >> C
    - 回合重複：A >> B >> C >> 　A >> B >> C >>　 A >> B >> C
- 連續運動限制

    無限制：至少須建立一日課表。

    不可連續兩天：至少須建立一、二、三日的基本動作課表。

- 休息訊息
    - 至少需要設定 休息訊息01，最多可設定三個。若有多個休息訊息，會全部提供給前端使用。
    - 文字欄位為必填，音檔不限制。
- 運動強度
    - 計算設定

        勾選計秒或者計次，會影響下方強度設定欄位的啟用狀態。

        如果動作設定計次，但在課程設定的動作強度未設定計次的資料，會造成前端運行運動時有問題，但系統無法報錯。

    - 強度分級

        分為固定強度、三級、五級，影響下方強度設定欄位。

    - 間歇休息時間

        動作與動作之間的休息時間，預設 0 秒。

    - 強度設定
        - 可設定計次與計秒情況下每個動作的運動組數及次數。
        - 每個強度至少都需要設定一個背景音樂。

- 訓練課表
    - 有設定連續限制的課程，須設定一二三日的基本動作課表才能通過資料驗證。
    - 沒有連續限制的課程，僅須設定一日基本動作課表。
- 動作列表

    分為基本動作及進階動作列表。

    課程資訊頁面保存時並沒有驗證動作數量，但是在設定課表頁面內有驗證，沒有動作會無法新增課表。
