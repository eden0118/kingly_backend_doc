# 課表欄位說明

- 課表天數：可選擇一到三日，不可重複建立。（前端已做判斷無法選擇）
- 動作難度

    可選擇基礎動作或進階動作。

    若所選擇的動作清單內動作數量為零，保存時會回報無法新增/儲存課表。

- 每日動作數量
    - 影響下方課程預覽建立產生的課表
    - 數量限制：此設定僅在 APP 內作用，限制使用者新增刪除動作的調整空間。
- 重複限制

    兩天以上的課表才可選擇。
    - 動作不重複
    - 部位不重複：選擇部位不重複時，若有動作設定多個部位，任一部位皆不可重複。

- 動作排序
    - 隨機排序

        按照上方欄位的限制設定，系統自動排列動作清單，有一定隨機性。

    - 固定排序

        需點擊旁邊課表預覽建立，會按照這裡所顯示的動作清單提供給使用者。

### 設定固定排序課表

1. 填寫基本資訊
2. 選擇動作難度
3. 填寫動作數量（固定排序基本上都需要人工調整，所以這邊隨便寫數字即可）
4. 設定重複限制
5. 動作排序選擇固定排序

### 設定隨機排序課表

1. 填寫基本資訊
2. 選擇動作難度
3. 填寫動作數量
4. 設定重複限制
5. 動作排序選擇隨機排序
