# 編輯部位

1. 找到要編輯的部位
   ![alt text](img/edit-body-part-01.png)
2. 點選編輯
   ![alt text](img/edit-body-part-02.png)
3. 修改名稱/說明，注意名稱不可與目前已有的部位重複
   ![alt text](img/edit-body-part-03.png)
4. 可修改或新增多語系設定；部位說明可透過欄位上方之語系切換按鈕（ZH/CH/EN，預設語系必填）進行修改各語系內容。
5. 修改完畢點擊保存
   ![alt text](img/edit-body-part-04.png)
6. 更新成功
   ![alt text](img/edit-body-part-05.png)
