# 數據評比項目

數據評比內依照需求可建立不同類別的評比項目，每個評比項目需要建立對應的數據表格。

## 操作流程

### 新增評比項目

- 進入評比版本後，點選新增
  ![alt text](img/add-transform-subject-01.png)

- 新增評比項目頁面分為3個區塊
  :::warning 必填欄位
  此頁面若有任何一個空缺欄位即無法送出。
  :::
    1. 基本資料
    2. 示意圖
    3. 對應級距設定：使用者填寫的數值對應的強度，這邊務必注意填寫數據的正確性，如有填錯會導致判斷邏輯失誤。

### 編輯評比項目

- 進入評比版本後，點選 編輯
  ![alt text](img/edit-transform-subject-01.png)

-

### 刪除評比項目

- 點擊刪除
  ![alt text](img/delete-transform-subject-01.png)

- 再次確認即刪除該評比項目，此刪除無法復原，請謹慎操作。

    ![alt text](img/delete-transform-subject-02.png)
