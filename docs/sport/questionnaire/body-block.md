# 身體數據區塊

> 這個區塊主要是詢問使用者基本身體數據以及肌耐力／柔軟度調查，以便後續評估給予運動強度建議。

## 進入區塊設定

- 點選 區塊設定
  ![alt text](img/body-block-01.png)
- 進入後看到此頁包含三個區塊:
    1. 基本資訊：可調整區塊名稱與補充說明，並新增多語系設定（ZH/CH/EN，預設語系必填）。
    2. 問題列表：已設定的問題列表。
    3. 設定數據評比標準，每個問卷可以綁定一個版本的評比標準。

    ![alt text](img/body-block-02.png)

### 新增／編輯／刪除問題

- 問題的設定操作同 [編輯一般區塊](./normal-block.md)。
- 列表內原則上可新增任意問題，注意先了解評比標準有哪些項目，避免詢問重複問題，這裡詢問的問題僅作為紀錄，無法做後續的判斷。

### 綁定數據評比標準

這裡可綁定數據評比標準，數據評比的設定參考 [數據評比結果](./transform-version.md)。

![alt text](img/body-block-03.png)

依照下拉選單選擇的版本，右側會顯示該版本內的評比項目。同個評比版本可以被多個問卷綁定。
![alt text](img/body-block-04.png)
