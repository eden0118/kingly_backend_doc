# 平台基本介紹與說明

運動平台的後台系統，除了共用功能外，還需管理與運動相關的專屬功能，主要支援運動百科網站的內容建置與 APP 課程。

## 功能項目

### 共用功能

- 網站設定
- 主題內容管理
- 媒體庫
- 關鍵字管理
- 廣告管理

> 參考 [共用平台功能說明](../base/indro.md)。

### 運動平台專屬功能

- 健康目的管理
- 健康問卷管理
- 器材管理
- 部位管理
- APP 課程管理

## 維持運作需建立的基礎 Database

- 健康問卷及數據評比
    - 系統預設開 1.0 版本，內容為空白，需完善內容並通過資料驗證才能啟用。
- 建立器材
- 建立部位
- APP 課程資料
    - 建立運動項目
    - 運動項目之下建立課程
    - 課程內完善資料
        1. 基本欄位
        2. 運動強度設定
        3. 新增動作（需先確認部位是否已設定）
        4. 建立課表（有動作才能建立課表）
- 建立健康目的，並設定目的對應課程
