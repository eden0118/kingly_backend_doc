# 推播運動

## 新增推播運動

1. 點擊側邊欄`情境推播管理`進入推播運動管理頁面。
2. 推播運動管理頁面會顯示目前已建立推播運動列表。
3. 右上角點擊`新增`。
4. 新增推播運動頁面，填寫推播時段等基本資訊。
    - 動作名稱：必填，注意名稱不可與已有的情境名稱重複。
    - 多語系設定：可新增簡體中文或英文的動作名稱。
    - 推播時段：必填，可勾選推播時段(上午、中午、晚上)
    - 主要推播時段：必填，從已勾選的推播時段中選擇一個作為主要推播時段。
    - 新增情境：請依照現有資料庫搜尋並輸入情境；情境管理相關操作請參考[情境管理](Contextual-management.md)。
    - 背景音樂：必填，有檔案格式限制
        - 格式限制：mp3,wav
        - 檔案大小限制：6 MB
    - 上傳動作影片（直式）：必填，有檔案格式限制。
        - 格式限制：mp4
        - 大小限制：30 MB
        - 尺寸限制：1080 x 1920
5. 點擊`新增`。
6. 新增成功。

## 編輯推播運動

1. 找到要編輯的推播運動名稱。
2. 點選`編輯`，進入編輯頁面。
3. 修改名稱/新增多語系設定，注意名稱不可與目前已有的情境重複。
4. 視需求調整推播時段與對應情境。
5. 依格式限制修改背景音樂/動作影片。
6. 修改完畢點擊`保存`。
7. 更新成功。

## 刪除推播運動

1. 找到欲刪除的推播名稱
2. 點選 `刪除`
3. 再次確認刪除
4. 刪除成功
