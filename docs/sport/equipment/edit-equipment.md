# 編輯運動器材

1. 找到要編輯的器材
   ![alt text](img/edit-equipment-01.png)
2. 點選編輯
   ![alt text](img/edit-equipment-02.png)
3. 修改名稱/說明，注意名稱不可與目前已有的器材重複
   ![alt text](img/edit-equipment-03.png)
4. 可修改或新增多語系設定；器材說明可透過欄位上方之語系切換按鈕（ZH/CH/EN，預設語系必填）進行修改各語系內容。
5. 修改完畢點擊保存
   ![alt text](img/edit-equipment-04.png)
6. 更新成功
