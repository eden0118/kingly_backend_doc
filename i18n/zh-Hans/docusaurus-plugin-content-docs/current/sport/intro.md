# 平台基本介绍与说明

运动平台的后台系统，除了共用功能外，还需管理与运动相关的专属功能，主要支援运动百科网站的内容建置与 APP 课程。

## 功能项目

### 共用功能

- 网站设定
- 主题内容管理
- 媒体库
- 关键字管理
- 广告管理

> 参考 [共用平台功能说明](../base/indro.md)。

### 运动平台专属功能

- 健康目的管理
- 健康问卷管理
- 器材管理
- 部位管理
- APP 课程管理

## 维持运作需建立的基础 Database

- 健康问卷及数据评比
- 系统预设开 1.0 版本，内容为空白，需完善内容并通过资料验证才能启用。
- 建立器材
- 建立部位
- APP 课程资料
- 建立运动项目
- 运动项目之下建立课程
- 课程内完善资料
    1. 基本栏位
    2. 运动强度设定
    3. 新增动作（需先确认部位是否已设定）
    4. 建立课表（有动作才能建立课表）

- 建立健康目的，并设定目的对应课程
