# 新增部位

> 部位用于动作设定内使用，主要影响部分课程有部位不可重复的限制；另外预留之后可能会有展示部位说明／示意图的需求。

1. 点击侧边栏 部位管理 进入 部位列表
   ![alt text](img/add-body-part-01.png)
2. 部位列表显示部位名称及已关联的动作
   ![alt text](img/add-body-part-02.png)
3. 右上角点击 新增部位
   ![alt text](img/add-body-part-03.png)
4. 填写 部位名称/说明。注意名称不可与已有的部位重复。
   ![alt text](img/add-body-part-04.png)
5. 点击 新增
   ![alt text](img/add-body-part-05.png)
6. 新增成功
