# APP 課程資料結構說明

課程結構：運動項目 → 課程 → 課表 → 動作
![alt text](img/course-info-01.png)
運動分為兩個類型：熱身運動及一般運動，主要差別在於熱身運動不會進入課程推薦清單內，且熱身運動就是一個運動項目，課程內只需要一日課表；一般運動分為多個運動項目，每個項目內可增加多個課程。
![alt text](img/course-info-02.png)

## 限制說明

- 無連續限制的課程的課程至少須建立一日課表，有連續限制的課程需要建立一到三日的課表，否則課程沒有完整資料，在健康目的那裏無法被設定對應關係。　參考 [運動課程](./course-manage.md)。
- 必須先有動作才能設定課表。　參考 [運動課表](./schedule.md)。
